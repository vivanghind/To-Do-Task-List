# 📝 Task List Application

A responsive Task Management web application built using **HTML**, **JavaScript**, and **Materialize CSS**. The application allows users to efficiently organize and manage their daily tasks with features such as task priorities, due dates, filtering, and local storage persistence.

---

## 🚀 Features

- ➕ Add new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete individual tasks
- 🧹 Clear all tasks with confirmation
- 📅 Assign due dates using a date picker
- 🚩 Set task priority (High, Medium, Low)
- 🔍 Search tasks by name
- 🎯 Filter tasks by priority
- 💾 Automatically save tasks using Local Storage
- 📱 Responsive UI built with Materialize CSS
- ⚠️ Confirmation modals for delete operations

---

---

## 📸 Features Overview

- Add tasks with a due date and priority.
- View all saved tasks.
- Search tasks using keywords.
- Filter tasks based on priority.
- Delete individual tasks or clear the entire list.
- Tasks remain saved even after refreshing the page using Local Storage.

---

## 💾 Data Storage

This application uses the browser's **Local Storage** to store tasks.

- No backend or database is required.
- Data is stored locally in the user's browser.
- Clearing browser storage will remove all saved tasks.
